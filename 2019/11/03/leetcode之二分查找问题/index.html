  • 存在着两段性的性质，即一半满足某一个性质，另一半不满足某个性质。

均可以使用二分查找。

红色表示满足某一个性质，绿色表示不满足某一个性质。

模板一

求红色的右边界情况。

if mid in 红 [L,R]->[M,R] L=M

else mid in 绿 [L,R]->[L,M-1] R=M-1

注意，mid应该是取(L+R)/2+1，防止死循环

else mid in 红 [L,R]->[M+1,R] L=M+1
