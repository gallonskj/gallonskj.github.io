leetcode之二分查找问题

发表于 2019-11-03 | 分类于 leetcode

对于O(logn)以及有序数组处理的问题，极有可能使用二分查找。

二分查找框架

使用二分查找算法还是有许多小问题的，比如说是否应该带等号，mid 是否应该加一等等。

while循环里有return语句带=，没有return语句不用带。

基本的二分查找

搜索一个数。

69.求开方

744.寻找比目标字母大的最小字母

540.有序数组的 Single Element

153.旋转数组的最小数字

34.在排序数组中查找元素的第一个和最后一个位置

总结

关于=的问题。

当while循环中，start = mid+1,end = mid-1这种，必定需要加等号。

当while循环中，有start = mid或者是end=mid，不加等号。
